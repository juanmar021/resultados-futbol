# Resultados de futbol

Muestra los resultados de fubtol sumistrados por https://rapidapi.com/api-sports/api/api-football
Asi como tambien el resultado de un partido en vivo.

Uso facil de:

-StreamBuilder
-Peticiones http
-Patron Bloc
-SharedPreferences
